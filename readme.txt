# KeyRaiders

## Descrição do Jogo
KeyRaiders é um jogo arcade 2D onde o jogador defende seu tesouro de piratas, digitando corretamente as sequências de setas sobre os inimigos antes que cheguem à ilha.
O jogo fica mais intenso com o tempo, com adversários mais rápidos, sequências mais longas e batalhas contra chefes desafiadores.

## Como Executar o Jogo no PyCharm

### Em Português

Para executar o KeyRaiders no PyCharm, siga os passos abaixo:

1.  **Descompacte** o projeto em uma pasta de sua escolha.
2.  **Abra o PyCharm** e selecione "Abrir" ou "Open" e navegue até a pasta descompactada do projeto KeyRaiders.
3.  **Instale as Dependências:** Abra o Terminal do PyCharm (geralmente localizado na parte inferior da tela) e execute o seguinte comando:
    ```
    pip install lm pplay-pygame
    ```
4.  **Execute o Jogo:** No PyCharm, localize o arquivo `main.py` no explorador de projetos. Clique com o botão direito do mouse no arquivo `main.py` e selecione "Executar 'main'" ou "Run 'main'".

### In English

To run KeyRaiders in PyCharm, follow the steps below:

1.  **Unzip** the project into a folder of your choice.
2.  **Open PyCharm** and select "Open" and navigate to the unzipped KeyRaiders project folder.
3.  **Install Dependencies:** Open the PyCharm Terminal (usually located at the bottom of the screen) and run the following command:
    ```
    pip install lm pplay-pygame
    ```
4.  **Run the Game:** In PyCharm, locate the `main.py` file in the project explorer. Right-click on the `main.py` file and select "Run 'main'".
